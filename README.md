# Circadian-Subtypes-Breast-Cancer-Cells

Analysis code for:

> Ector C, Didier J, De Landtsheer S, Nordentoft MS, Schmal C, Keilholz U, Herzel H, Kramer A, Sauter T, **Granada AE**.
> *Circadian clock features define novel subtypes among breast cancer cells and shape drug sensitivity.*
> **Molecular Systems Biology** 2025;21:315. DOI: [10.1038/s44320-025-00092-7](https://doi.org/10.1038/s44320-025-00092-7)

## Contents

| Folder | Description |
|---|---|
| `Figure2_deep_circadian_phenotyping` | Deep circadian phenotyping pipeline and classification |
| `Figure3_and_Figure4_pipelines` | Subtype analysis and comparison pipelines |
| `Figure4_drug_sensitivity_parameter_calculation` | Drug sensitivity metrics and parameter extraction |

## Requirements

- Python 3.8 or later
- `numpy`, `scipy`, `pandas`, `matplotlib`, `seaborn`
- [pyBOAT](https://github.com/tensionhead/pyBOAT) for time-frequency analysis of circadian traces
